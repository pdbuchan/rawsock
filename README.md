# rawsock
C Language Examples of IPv4 and IPv6 Raw Sockets for Linux<br/><br/>

I have recently been investigating raw socket programming in C for linux and I decided to provide a collection of routines I have prepared. The intention here is to be able to arbitrarily set the various parameters within a packet.<br/><br/>
Rather than use command-line arguments, each example has hard-coded values, so you need to modify each example to suit your preferences.<br/><br/>
You must run these as root to obtain raw sockets. You don't need pcap.
